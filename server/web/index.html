      Supported file types

      Drop any of these into a corpus via Add doc. Images, archives, and other binaries are skipped automatically.

      • .pdf — PDF documents (indexed per page)
      • .docx .doc — Word documents
      • .txt .md .markdown .rst — plain text / Markdown
      • .csv .log — tabular / log text
      • .json .xml .html .htm — structured / markup text
