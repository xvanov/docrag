North Carolina

About

The entirety of the 2018 North Carolina State Building Code including the amendments remains effective.

In the interim, pursuant to Section 102.5 of the 2018 North Carolina Administrative Code and Policies, the 2024 North Carolina State Building Code may be used as an alternate method of construction at the building owner’s or his agent’s request.

Until the to-be determined effective date of the 2024 North Carolina State Building Code, the 2018 North Carolina State Building Code remains in full force and should continue to be enforced as the governing standard, unless a building owner or his agent requests that the 2024 North Carolina State Building Code be applied to the project.

North Carolina has mandatory statewide code enforcement. The North Carolina Building Code Council is the authority that oversees the code process. The Residential Code is on a 6-year adoption cycle by statute. The Council has also adopted a policy that all other codes except the NEC on a 6-year cycle to match the residential code. Local amendments include:

  • 2018 NC State Building Code Amendments
  • 2018 NC State Residential Code Amendments
  • 2018 NC State Energy Conservation Code Amendments
  • 2018 NC State Existing Building Code Amendments
  • 2018 NC State Fire Code Amendments
  • 2018 NC State Fuel Gas Code Amendments
  • 2018 NC State Mechanical Code Amendments
  • 2018 NC State Plumbing Code Amendments
  • 2020 North Carolina State Electrical Code
